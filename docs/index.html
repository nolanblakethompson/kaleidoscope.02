GENERATIVE KALEIDOSCOPE.02

GENERATIVE KALEIDOSCOPE.02 is a p5.js based tool that creates a kaleidoscope through digitally drawn lines. Follow the direcitons below to help me collect data on the functionally!

1 >> Press + hold + drag to draw a line
2 >> draw your name ( do not try and make it look like your name, simply draw your name and let the code do its job! )
3 >> press save to save the file as a png
4 >> upload your png to this google drive folder :)
